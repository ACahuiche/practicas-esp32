# Curso de Electrónica Práctica

Repositorio con el código de mis prácticas del curso de 
electrónica desde cero hasta nivel intermedio-avanzado.

## Contenido

### ESP32
| Práctica | Descripción |
|----------|-------------|
| [practica1-toggle-button](https://github.com/ACahuiche/practicas-esp32/tree/main/toggle_button) | Toggle con Bounce2 |
| [practica2-potenciometro-led](https://github.com/ACahuiche/practicas-esp32/tree/main/lectura_analoga) | Control de brillo con potenciómetro y map() |
| [practica3-potenciometro-dos-leds](https://github.com/ACahuiche/practicas-esp32/tree/main/potenciometro-dos-leds) | Control de dos LEDs con potenciómetro, map() y constrain() |
| [practica4-dht11-leds-temperatura](https://github.com/ACahuiche/practicas-esp32/tree/main/lector_dht11) | DHT11 con LEDs indicadores de temperatura |
| [practica5-hcsr04-distancia](https://github.com/ACahuiche/practicas-esp32/tree/main/sensor_ultrasonico) | HC-SR04 con LEDs indicadores de proximidad |
| [practica6-oled-contador](https://github.com/ACahuiche/practicas-esp32/tree/main/contador_oled) | Pantalla OLED SSD1306 con contador 0-100 |
| [practica7-oled-dht11-temperatura](https://github.com/ACahuiche/practicas-esp32/tree/main/lector_temperatura_oled) | Termómetro ambiental con OLED y DHT11 |
| [practica8-pir-detector-movimiento](https://github.com/ACahuiche/practicas-esp32/tree/main/sensor_movimiento) | PIR SR501 con contador de movimientos |
| [practica9-sensor-tactil-ttp223b](https://github.com/ACahuiche/practicas-esp32/tree/main/sensor_tactil) | TTP223B con ciclo de brillo por toque |

## Hardware
- ESP32 WROOM 32 de 38 pines
- Arduino IDE con esp32 by Espressif Systems

## Simulación
- Wokwi para simulaciones previas
- Tinkercad para electrónica básica (parte 1)
