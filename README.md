# Curso de Electrónica Práctica

Repositorio con el código de mis prácticas del curso de 
electrónica desde cero hasta nivel intermedio-avanzado.

## Contenido

### ESP32
| Práctica | Descripción |
|----------|-------------|
| practica1-toggle-button | Toggle con Bounce2 |
| practica2-lectura-analoga | Control de brillo con potenciómetro y map() |
| practica3-potenciometro-dos-leds | Control de dos LEDs con potenciómetro, map() y constrain() |
| practica4-dht11-leds-temperatura | DHT11 con LEDs indicadores de temperatura |
| practica5-hcsr04-distancia | HC-SR04 con LEDs indicadores de proximidad |
| practica6-oled-contador | Pantalla OLED SSD1306 con contador 0-100 |

## Hardware
- ESP32 WROOM 32 de 38 pines
- Arduino IDE con esp32 by Espressif Systems

## Simulación
- Wokwi para simulaciones previas
- Tinkercad para electrónica básica (parte 1)
